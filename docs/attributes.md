# Attribute `maxFileSize`

------------


The base mirror event for all Mirror events that `Draggable` emits.

| attribute | |
| --------------------- | ---------------------------------------------------------- |
| **Interface**         | `MirrorEvent`                                              |
| **Cancelable**        | false                                                      |
| **Cancel action**     | -                                                          |
| **type**              | `mirror`                                                   |
