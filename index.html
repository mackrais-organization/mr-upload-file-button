Description

Plugin for ajax uploading files. Development using the jQuery. Very easy to use and easy to modify for any requirements. The advantages of this plugin are that it is very fast, flexible and also free. There is also a full documentation and many examples of how to use this plugin.

Features

  • Drag and Drop area.
  • Easily customize to fit your needs.
  • You can use both a separate widget or part of the form.
  • Previews to images, (extensions to file)
  • Own gallery to view images
  • Support multi language
  • Sortable thumbnail
  • Remove thumbnail
  • Ajax upload
  • Mobile device gallery support. Needed jQueryMobile
